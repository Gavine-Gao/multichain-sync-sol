Centralized Solana Wallet Middleware is one of the DappLink One-click deployment of wallets Components.

**Note**: Requires [Go 1.22+](https://golang.org/dl/)

## Installation

For prerequisites and detailed build instructions please read the [Installation](https://github.com/Gavine-Gao/multichain-sync-sol/) instructions. Once the dependencies are installed, run:

```bash
make multichain-sync
```

Or check out the latest [release](https://github.com/Gavine-Gao/multichain-sync-sol).
